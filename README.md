💈 Barbearia Prime

Projeto desenvolvido para a disciplina de Desenvolvimento de Sistemas, com foco na criação de uma página web moderna, responsiva e bem estruturada.

----------------------------------------------------------------------------

📌 Sobre o Projeto

A Barbearia Prime é um site institucional que apresenta os serviços de uma barbearia premium, destacando estilo, tradição e experiência do cliente.
O projeto foi desenvolvido utilizando HTML e CSS, com apoio do Bootstrap para componentes interativos.

-----------------------------------------------------------------------------

🚀 Tecnologias Utilizadas

- HTML5  
- CSS3  
- Bootstrap 5  

-----------------------------------------------------------------------------

🎯 Funcionalidades

✔ Menu responsivo (mobile com botão hamburguer)  
✔ Navegação por seções (scroll suave)  
✔ Banner inicial com chamada de ação  
✔ Seção "Sobre" com descrição da barbearia  
✔ Carousel de serviços com imagens e descrição  
✔ Tabela de preços organizada por categorias  
✔ Galeria de imagens  
✔ Seção de contato com informações completas
